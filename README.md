## 🔮 Sobre o projeto

Este projeto consiste em um site desenvolvido para apresentar e facilitar o agendamento de consultas de **tarot terapêutico**, oferecendo uma experiência digital imersiva, acolhedora e intuitiva.

A proposta é conectar estética, espiritualidade e tecnologia em um único ambiente, criando uma jornada fluida para o usuário.

---

## 🌙 Objetivo

O objetivo é oferecer uma experiência clara e envolvente para os visitantes, reunindo em um único lugar:

▫️ Apresentação do trabalho da terapeuta  
▫️ Explicação sobre como funcionam as consultas  
▫️ Informações sobre valores e serviços  
▫️ Canal direto para contato e agendamento  
▫️ Espaço para depoimentos e prova social

## ⚙️ Funcionalidades

▫️ Página inicial com identidade visual imersiva  
▫️ Seção "Sobre" com apresentação da terapeuta  
▫️ Página "Como funciona" explicando o processo  
▫️ Página de valores e serviços  
▫️ Depoimentos de clientes  
▫️ Formulário de contato  
▫️ Integração direta com WhatsApp  
▫️ Sistema de mensagens com atualização em tempo real  
▫️ Área administrativa para gerenciamento de depoimentos

## 🔐 Segurança e controle

O sistema possui algumas proteções básicas:

▫️ Validação de campos (nome e mensagem)  
▫️ Limite de envio para evitar spam  
▫️ Controle de acesso para ações administrativas  
▫️ Exclusão de mensagens restrita ao administrador
